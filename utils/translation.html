Buongiorno Mariagrazia,

inizialmente Cedacri si aspettava la ricezione del file in codifica EBCDIC; di conseguenza, quando il file veniva trasmesso in formato ASCII, non risultava correttamente leggibile dai loro sistemi.

Successivamente, Cedacri ha adeguato i propri sistemi per supportare anche la ricezione dei file in codifica ASCII e, da quel momento, la problematica non si è più verificata.

Attualmente, in ambiente PROD, il file viene trasmesso in codifica ASCII.
